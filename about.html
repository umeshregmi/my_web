git init git config --global user.name 'your_username' git config --global user.email 'your_email' git status --> checks the status of file git add file_name (or . to add all file) --> adds file to staging area git commit -m 'your commit message' --> commits the file git push origin branch_name --> pushes code to remote repository git branch --> lists all branch name git branch branch_name --> creates a new branch git checkout branch_name --> switch branch git log --> lists all the commit logs git pull origin branch_name
